Jumping = Shooting = Jumping

About

Vice Versa is a 2D arena shooter where your gun can only shoot downward when touching ground, but doing so blasts you upward, letting you shoot anywhere while airborne. Your objective is to purge an endless wave of spirits in a dream-like world and survive the longest you can.

  • Game page: Itch.io link
  • Released: 2021
  • Engine: GameMaker 2
  • Platforms: Windows

Vice Versa is a solo project. As a big fan of Sorath's Devil Daggers, I began working on this game as a prototype of how that game could translate into 2D. It eventually turned into something different, but it retained some similarities as I enjoy that game a lot and I wanted to show my appreciation for it.
